# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-22

### Added

- Initial open-source release of Discharge
- Service-oriented plugin architecture for triggers, runners, and VCS integrations
- GitHub Issues trigger with label-based filtering and comment triggers
- Sentry trigger for error tracking integration
- CircleCI trigger for CI/CD failure handling
- Claude Code runner for AI-powered bug investigation and fixing
- GitHub VCS integration with GitHub App authentication
- Conversation mode for multi-turn AI interactions
- Plan approval workflow with human-in-the-loop
- Web UI for managing projects, viewing conversations, and monitoring jobs
- BullMQ-based job queue with Redis backend
- PostgreSQL database with Drizzle ORM
- Docker support for development and production
- Comprehensive test suite with unit and integration tests
