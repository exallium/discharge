# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Pledge

We pledge to make participation in our community a harassment-free experience for everyone.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers at the email address listed in the project's GitHub profile.

All complaints will be reviewed and investigated promptly and fairly.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
